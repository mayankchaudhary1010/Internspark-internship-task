# Zomato Data Analysis

## Objective
Analyze restaurant data to extract insights on ratings, cuisines, and pricing.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Most ratings are between 3.5–4.2
- Popular cuisines dominate demand
- Price does not strongly affect ratings
